# futurejob
단순한 직업 리스트를 넘어, 미래 사회의 청사진을 보여주는 종합적인 데이터베이스입니다.

진로 설계자는 본인의 흥미(RIASEC)와 성격(BIG5)을 고려하여 유망 직업을 탐색하고, 필요한 핵심 지식과 준비 기간을 파악하여 장기적인 계획을 세울 수 있습니다.
교육 정책 입안자는 미래 산업 수요에 맞춘 교육 커리큘럼을 개발하고, '상', '중', '하' 등급별로 필요한 인재 양성 정책을 수립하는 데 활용할 수 있습니다.
연구자/기획자의 경우 특정 산업 분야의 미래 직업 변화 트렌드를 분석하고, 기술 발전이 노동 시장에 미칠 영향을 예측하는 기초 자료로 사용할 수 있습니다.
